🏙️ Evolution City Beta 🏙️

Un gioco gestionale sviluppato interamente con **HTML5, CSS3 e JavaScript Vanilla**, ispirato ai grandi classici dei city builder.

Costruisci la tua città, gestisci il denaro, aumenta la popolazione e sviluppa un'economia sostenibile. Il progetto è stato realizzato con un'architettura modulare, facilmente espandibile con nuove funzionalità come energia, acqua, traffico, meteo e missioni.

🎯 Obiettivo

Espandere una piccola città fino a trasformarla in una grande metropoli amministrando con attenzione le risorse disponibili.

Ogni edificio possiede un costo e, in alcuni casi, genera entrate automatiche che consentono di finanziare nuove costruzioni.

---

🏗️ Edifici disponibili

| Edificio      | Costo |                       Produzione |
| ------------- | ----: | -------------------------------: |
| 🏠 Casa       | 100 € |                      +5 abitanti |
| 🏢 Ufficio    | 350 € |                        +15 €/sec |
| 🏭 Fabbrica   | 600 € |                        +40 €/sec |
| 🌳 Parco      |  80 € |        Incremento qualità urbana |
| 🛣️ Strada    |  20 € |                   Infrastruttura |
| 🗑️ Bulldozer |     — | Demolizione con rimborso del 50% |

---

💰 Economia

Il gioco aggiorna automaticamente le entrate ogni secondo.

* Le case aumentano la popolazione.
* Gli uffici producono reddito.
* Le fabbriche generano un'economia maggiore.
* La demolizione restituisce il 50% del costo dell'edificio.

---

💾 Salvataggio

La partita viene salvata automaticamente tramite **LocalStorage**.

Sono disponibili anche:

* Salvataggio manuale
* Ripristino automatico
* Reset della città

🚀 Installazione

1. Clona il repository.

```bash

```

2. Apri la cartella del progetto.

3. Avvia `index.html` nel browser.

Non sono richieste librerie o framework esterni.
